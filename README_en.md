## Shortcuts

### Shortcut to open Bean selection dialog

Mac: `Command + Shift + I`

Windows: `Ctrl + Shift + O`

### Shortcut to open injection settings (only available when Bean selection dialog is active)

Mac: `Option + Enter`

Windows: `Alt + Enter`
